Clinic Appointment & Inventory Management System (CAIMS)

A functional C++ console based clinic system built to support common clinic operations such as managing appointments, patient information, and inventory records. The project focuses on creating a structured workflow where users can log in, access menu options, and handle clinic data in a more organized way.

  • Built with C++ using a menu based console interface.
  • Includes login flow for basic access control.
  • Supports clinic related records such as appointments and inventory.
  • Designed to make daily clinic management more structured and efficient.

Database Management System

A MySQL database project created to store and organize clinic related information for the CAIMS system. The database was designed using multiple connected tables so that appointments, patients, medicines, prescriptions, medical certificates, and users could be managed in a clear and structured way.

  • Designed database tables for clinic operations and records.
  • Used phpMyAdmin to manage and inspect the database structure.
  • Organized data into separate tables for better clarity and maintenance.
  • Focused on making stored information easier to search, update, and manage.
